
1️⃣ What is the difference between var, let, and const?  
- `var` → function scoped, can redeclare  
- `let` → block scoped, can change value  
- `const` → block scoped, cannot change value  

2️⃣ What is the spread operator (...)?**  
- Copies items from arrays or objects easily  
```js
let arr2 = [...arr, 3];

3️⃣ What is the difference between map(), filter(), and forEach()?

map() → returns new array after change

filter() → returns new array after condition

forEach() → loops through items, no new array
